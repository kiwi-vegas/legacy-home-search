# Market Reports Pipeline — Legacy Home Search

## What It Is

A monthly automated pipeline that turns Altos Research PDF data into full market reports published on the site. Barry downloads the Altos PDF for each city, uploads it to the admin page, Claude reads the PDF natively and writes a complete 5-section report in Barry's voice, and the report is published to `/market-reports/[slug]` and appears in the `/blog` listing.

---

## End-to-End Flow

1. **Barry logs into Altos Research** → downloads the monthly PDF for each city
2. **Barry visits** `/admin/market-reports/upload?secret=ADMIN_SECRET`
3. **Selects city + uploads PDF** → Claude reads it natively (no PDF parser) and writes the full report
4. **Cover image generated** via OpenAI DALL-E 3 (YouTube/financial illustration style)
5. **Draft saved to Sanity** (`published: false`)
6. **Barry receives email** → "Your Virginia Beach April 2026 report is ready — review and publish with one click"
7. **Barry visits** `/admin/market-reports/review/[id]?secret=ADMIN_SECRET` → reads, optionally edits, clicks Publish
8. **Report goes live** at `/market-reports/[slug]` (ISR revalidates in 60 seconds)
9. **Report also appears** in `/blog` listing (merged with regular posts, sorted by date)
10. **Community pages** (`/virginia-beach`, `/chesapeake`, etc.) show a "Latest Market Report" card above the Altos iframe

---

## Admin URLs

- **Easy upload link**: `https://legacy-home-search.vercel.app/upload`
  - Enter password once → 7-day session cookie → no password needed for 7 days
  - Password = `ADMIN_PIN` env var (short memorable value you set); falls back to `ADMIN_SECRET` if not set
- **Direct upload page**: `/admin/market-reports/upload?secret=ADMIN_SECRET`
- **Review page**: `/admin/market-reports/review/[draftId]?secret=ADMIN_SECRET`
  - Link is emailed automatically after each upload

## Sanity Schema: `marketReport`

Document type fields:
- `community` — slug key: `virginia-beach`, `chesapeake`, `norfolk`, `suffolk`, `hampton`, `newport-news`
- `communityName` — display name: `Virginia Beach`
- `reportPeriod` — `April 2026`
- `slug` — `virginia-beach-april-2026`
- `published` — boolean; `false` until Barry clicks Publish
- `publishedAt` — set when published
- `altosEmailText` — raw email text (stored for reference, not displayed)
- `medianListPrice`, `medianPriceChange`, `daysOnMarket`, `activeInventory`, `inventoryChange`, `priceReductions` — extracted metric strings
- `marketSummary`, `buyerSection`, `sellerSection`, `investorSection`, `barrysTake` — Claude-written narrative sections
- `metaTitle`, `metaDescription` — SEO fields
- `coverImage` — Sanity image asset (generated by DALL-E 3)

## Claude PDF Reading

The `/api/market-reports/generate-from-pdf` route uses Claude's **native PDF document support**:

```typescript
{ type: 'document', source: { type: 'base64', media_type: 'application/pdf', data: base64 } }
```

No PDF parsing library needed — Claude reads the PDF directly. Model: `claude-sonnet-4-6`, `max_tokens: 4096`.

Claude extracts all stats from the PDF **and** writes all 5 narrative sections in one call. Returns JSON.

## Voice / Writing Style

Barry Jenkins persona:
- Lead agent, Legacy Home Team, 20+ years in Hampton Roads
- Team sells ~900 homes/year
- Audience: ordinary homeowners, first-time buyers, investors — plain English
- **Every claim tied to a specific number** from the Altos data
- No fluff, no "exciting opportunities" — straight market intelligence
- Barry's Take: first person (I/my/we), draws on pattern recognition from 20 years of cycles

---

## Cover Image Style

Generated by **DALL-E 3** via `lib/image-gen-market-report.ts`.

Style: **YouTube thumbnail meets financial media** (Bloomberg, CNBC energy)
- Bold, graphic composition — NOT cinematic real estate photography
- Financial illustration elements: trend arrows, price tags, bar charts, % indicators
- Dark navy or deep background with electric blue, white, and gold/yellow accents
- Community name and a headline metric visible as large graphical text
- Illustrative, semi-realistic — data visualization brought to life
- Prompt crafted by Claude first, then passed to DALL-E 3 (`1792x1024`, `hd` quality)

This is intentionally different from blog post covers (which use Gemini with cinematic real estate photography).

## Blog Integration

`/blog` fetches both `getBlogPosts(24)` and `getMarketReports(12)` in parallel, merges them sorted by `publishedAt` descending. Market report cards show "Market Report" badge and link to `/market-reports/[slug]`.

---

## Community Page Integration

All 6 community pages (`/virginia-beach`, `/chesapeake`, etc.) show a "Latest Market Report" card in the Market Trends section above the Altos iframe. The card queries `getLatestMarketReport(community)` from Sanity — renders nothing if no published report exists yet.

## Monthly Safety-Net Cron

`vercel.json` includes a monthly cron: `0 14 1 * *` (7 AM PT on the 1st of each month).

Route: `/api/cron/market-reports` — checks if reports arrived for all 6 cities this month. If any are missing, emails Barry a heads-up.

## Inbound Email Route (Fallback)

`/api/inbound/market-report` accepts Resend inbound webhook POSTs. This was the original pipeline approach — Altos emails contain only a button link (not the data), so the **PDF upload approach is the primary flow**. The inbound route remains for potential future use.

If used: verifies Svix signature → extracts `email_id` → fetches body from `https://api.resend.com/emails/receiving/{id}` → detects community → writes report via `writeMarketReport()`.
